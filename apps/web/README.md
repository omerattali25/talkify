## Getting Started

### Prerequisites

- Node.js 18+
- Yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/talkify.git
cd talkify
```

2. Install dependencies:
```bash
yarn install
```

3. Start the development server:
```bash
yarn dev
```

The application will be available at `http://localhost:5173`.
